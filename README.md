# Llama's Bastion Practice

A bastion practice map for Minecraft version 1.16.1

## Features
- Natural Terrain
- Block Processors
- Random Mobs
- Random Bastion Rotations  

## Customization
- Toggle Terrain
- Toggle Hoglins
- Toggle Natural Mobs
- Selectable Gold Block Chances
- Selectable Spawnpoint
