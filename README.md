> Bastion Practice for Minecraft 1.16.1 speedruns

- Natural Terrain
- Block Processors
- Random Mobs

- Toggle Terrain
- Toggle Hoglins
- Toggle Natural Mobs
- Selectable Gold Block Chances
- Selectable Spawnpoint
